Mis habilidades

A lo largo de mi carrera, he tenido la oportunidad de trabajar con una variedad de tecnologías que me han permitido desarrollar aplicaciones web eficientes y escalables. Aquí te presento algunas de las principales herramientas y lenguajes en los que me he especializado:

Frontend:

TailwindCSS: Me ayuda a crear interfaces atractivas y completamente personalizables de manera rápida y eficiente.
Bootstrap: Herramienta fundamental para el desarrollo de sitios web responsive y bien estructurados.
JavaScript y TypeScript: Lenguajes clave para crear interactividad en las aplicaciones web, optimizando tanto la experiencia de usuario como la lógica del lado del cliente.

Backend:

C#, Java, y PHP: Lenguajes que me permiten construir el lado servidor de aplicaciones robustas, garantizando la integración perfecta entre el frontend y backend.
SQL: Mi experiencia en bases de datos me permite manejar la persistencia de datos de manera segura y eficiente.

Otros:

GitHub: Plataforma que uso para control de versiones y colaboración en proyectos.
Cypress: Una herramienta clave para asegurar la calidad de las aplicaciones mediante pruebas automatizadas end-to-end.
